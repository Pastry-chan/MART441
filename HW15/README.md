# Reflection
Conceptual Understanding

There were many important things I learned about working in a 3D space with Three.js. I think the most important thing I learned was how to import 3D assets and how to animate them.
I made my window model rotate as well but I didn't keep it in because the sides were see-though and it looked a bit strange but knowing I can learn how to make my own assets and be able
to import and use them is super helpful. It is much different than working in a 2D canvas, when working with 3D elements I am used to always being able to see it, and change it directly without
needing to code anything so that was strange for me.


Technical Challenges

The most challenging part was actually centering the camera towards the model so you could see it fully. It took me a long time, I googled and watched many tutorials on youtube. On occassion
I did use Gen AI to help de-bug whenever I couldn't pinpoint it myself or figure out why my window is starting off upside down and half way off the screen.

Design & Intent

I kept it simple because I have some finals due this weekend as well as an intermediate Japanese script I have to write, and memorize so I had little to no time. However there still was
a goal. I absoultely love surreal media such as Don't Hug Me I'm Scared, Vivarium, MirrorMask, and I am really looking forward to the Exit 8 movie. I wanted to go for something like that,
something surreal and kind of off putting. When I had the window spinning I think it really added to it but I was worried it looked too weird. 

Growth & Iteration

If I had more time there is so much I would do. I would love to learn how to make 3D assets so I can import them and make my own little scene. I think that's what I am more
interested in exploring with Three.js is animating.
